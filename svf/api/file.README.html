origen_svf

An Origen plugin to render JTAG-based patterns in Serial Vector Format (SVF).

To use, simply add the plugin to your appâ€™s Gemfile and require in your top-level lib file:

Customizing the Output for SVF

This plugin will extend the OrigenTesters API to add a .svf? method to all testers, allowing you to do this in your code:

1unless tester.svf?
2  # Something not suitable for SVF output
3end

A common use case may be to customize the pin list:

1if tester.svf?
2  # Only include these pins in .svf files
3  pin_pattern_order :rstn, :trstn, :done, :fail, only: true
4else
5  # Full pin list
6end
