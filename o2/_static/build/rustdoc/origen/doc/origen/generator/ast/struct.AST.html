pub fn push_and_open(&mut self, node: Node) -> usize[src]

Push a new node into the AST and leave it open, meaning that all new nodes added to the AST will be inserted as children of this node until it is closed. A reference ID is returned and the caller should save this and provide it again when calling close(). If the reference does not match the expected an error will be raised. This will catch any cases of AST application code forgetting to close a node before closing one of its parents.
