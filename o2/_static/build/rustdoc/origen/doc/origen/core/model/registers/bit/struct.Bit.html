Fields

id: usizeregister_id: usizeoverlay: RwLock<Option<String>>overlay_snapshots: RwLock<HashMap<String, Option<String>>>state: RwLock<u8>

The individual bits mean the following: 0 - Data value 1 - Value is X 2 - Value is Z 3 - Bit is to be verified 4 - Bit is to be captured 5 - Modified, sets if bits [2:0] have been changed since the last reset

device_state: RwLock<u8>

The state we think the device has, only bits [2:0] are applicable. This is updated by reseting the register or executing a transaction.

state_snapshots: RwLock<HashMap<String, u8>>

The state of the bit at the last reset
