pub fn resolve_path(&self, user_val: Option<&String>, default: &str) -> PathBuf[src]

Resolves a directory/file path relative to the application's root. Accepts an optional 'user_val' and a default. The resulting directory will be resolved from:

  1. If a user value is given, and its absolute, this is the final path.
  2. If a user value is given but its not absolute, then the final path is the user path relative to the application root.
  3. If no user value is given, the final path is the default path relative to the root. Notes: A default is required, but an empty default will point to the application root. The default is assumed to be relative. Absolute defaults are not supported.
