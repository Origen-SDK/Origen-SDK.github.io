id: usizename: Stringparent_id: Option<usize>

The only one without a parent is the top-level DUT model

sub_blocks: IndexMap<String, usize>

All children of this block/model, which are themselves models

memory_maps: IndexMap<String, usize>

All registers owned by this model are arranged within memory maps

pins: IndexMap<String, usize>pin_groups: IndexMap<String, usize>pin_headers: IndexMap<String, usize>timesets: IndexMap<String, usize>address_unit_bits: u32

Represents the number of bits in an address increment between two consecutive addressable units within the block. Its value defaults to 8 indicating that the address offsets of its sub-blocks will be expressed as byte addresses. This attribute can be overridden by individual MemoryMaps defined within the block. Since this attribute is intrinsically linked to the address definitions made within the block, it cannot be overridden by instantiation.

offset: u128

The starting address of the block expressed in address_unit_bits of the parent block. This defaults to 0 but can be overridden by instantiation.
